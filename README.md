# php_sql_todoList
This is an academical project to implement php code with sql data base management into a Todo List app.

 
## Stack

php, sql, login system


  
## Table of contents

- php with MVC architecture
- Session storage
- php queries PDO
- Data fetching
- phpMyAdmin
- Relational database
